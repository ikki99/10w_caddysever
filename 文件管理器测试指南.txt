                      快速测试步骤
════════════════════════════════════════════════════════════════

【测试 1：多选和批量压缩】

步骤：
  1. 启动程序：caddy-manager-console.exe
  2. 访问 http://localhost:8989 并登录
  3. 点击"文件管理"标签
  4. 点击"多选模式"按钮
  5. 勾选几个文件
  6. 点击"压缩选中"
  7. 输入压缩文件名（如：test-archive）
  8. 确认

预期结果：
  ✓ 界面显示选中文件数量
  ✓ 生成 test-archive.zip 文件
  ✓ 文件列表自动刷新
  ✓ 退出多选模式

────────────────────────────────────────────────────────────────

【测试 2：解压 ZIP 文件】

步骤：
  1. 找到刚才创建的 test-archive.zip
  2. 文件旁边应显示 📦 图标
  3. 点击"解压"按钮
  4. 确认解压

预期结果：
  ✓ 创建 test-archive 文件夹
  ✓ 所有文件解压到该文件夹
  ✓ 保留原始文件结构

【测试 3：代码编辑器】

步骤：
  1. 创建一个测试文件或找到 .js/.txt/.html 文件
  2. 点击文件名或"编辑"按钮
  3. 在编辑器中修改内容
  4. 按 Ctrl+S 或点击"保存"
  5. 关闭编辑器

预期结果：
  ✓ 弹出编辑器窗口
  ✓ 显示文件内容
  ✓ 显示语言类型（如 JavaScript）
  ✓ Ctrl+S 快捷键工作
  ✓ 保存成功提示
  ✓ 自动创建 .backup 文件

快捷键测试：
  • Tab 键 → 插入 4 个空格
  • Ctrl+S → 保存文件

预期结果：

  ✓ 成功消息显示

                      性能测试
════════════════════════════════════════════════════════════════

【压缩性能】
  • 压缩 10 个小文件（<1MB） → 应该<1秒
  • 压缩 100 个文件 → 应该<5秒
  • 压缩大文件夹（>100MB） → 可能需要等待

【解压性能】
  • 解压小文件（<10MB） → 应该<2秒
  • 解压大文件（>50MB） → 可能需要等待

【编辑器性能】
  • 打开小文件（<1MB） → 立即打开
  • 打开大文件（5MB） → 可能稍慢
  • 超过 10MB 文件 → 应该提示错误

════════════════════════════════════════════════════════════════
                      已知限制
════════════════════════════════════════════════════════════════

编辑器：
  • 文件大小限制 10MB
  • 无语法高亮（纯文本）
  • 不支持二进制文件

压缩：
  • 仅支持 ZIP 格式
  • 大文件压缩可能较慢
  • 压缩文件保存在当前目录

解压：
  • 仅支持 ZIP 格式
  • 解压到同名文件夹
  • 已存在文件夹会覆盖
